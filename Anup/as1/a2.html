Additional Examples

Local Image Example

To use a local image, replace the placeholder URL with your image path:

<img src="images/my-photo.jpg" alt="My Photo">

Different Image Formats Supported

  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • GIF (.gif)
  • SVG (.svg)
  • WebP (.webp)

Link Attributes

  • target="_blank" - Opens link in new tab
  • title - Shows tooltip on hover
  • rel="noopener" - Security attribute for external links
